Service, ServiceImpl#

Spring н”„лЎңм қнҠёлҘј н•ҳл©ҙ мў…мў… кҙҖлЎҖм ҒмңјлЎң ServiceлҘј interfaceлЎң кё°лҠҘ лӘ…м„ёлҘј н•ң л’Ө ServiceImplм—җ кё°лҠҘмқ„ кө¬нҳ„н•ҳкІҢ лҗҳлҠ” Factory Patternмқ„ мӮ¬мҡ©н•ҳкІҢ лҗ©лӢҲлӢӨ.

interfaceлҠ” кё°лҠҘмқ„ м¶”мғҒнҷ”н•ҳм—¬ нҒҙлһҳмҠӨк°„ кІ°н•©лҸ„лҘј лӮ®м¶”м–ҙ мЈјкі , нҳ‘м—… мӢң м—…л¬ҙл¶„лӢҙлҸ„ мҡ©мқҙн•©лӢҲлӢӨ.
кІҢмһ„мңјлЎң мҳҲлҘј л“Өл©ҙ мҠӨнғҖнҒ¬лһҳн”„нҠём—җм„ң лӘЁл“  мң лӢӣмқҳ кё°ліём Ғмқё нҠ№м„ұ HP, мқҙлҸҷн•ҳкё°лҘј interfaceлЎң кё°лҠҘл§Ң лӘ…мӢңн•ҳкі  к°Ғк°Ғ л¶„м—…н•ҳм—¬ мң лӢӣм—җ лҢҖн•ң HPлӮҳ мқҙлҸҷмҶҚлҸ„лҘј кө¬нҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

н•ҳм§Җл§Ң мқјл°ҳм Ғмқё Spring мӣ№н”„лЎңм қнҠём—җм„ңлҠ” Service interfaceлҠ” 1:1 кө¬мЎ°мқё кІҪмҡ°к°Җ л§ҺмҠөлӢҲлӢӨ. л§Ңм•Ҫ нҷ•мһҘм„ұмқ„ кі л Өн•ң 1:Nмқҳ кІҪмҡ°м—җлҠ” interfaceлЎң к°ҖлҠ” кІғмқҙ мўӢм§Җл§Ң л„Ҳл¬ҙ л§үм—°н•ң кІҪмҡ°м—җлҠ” к·ёлғҘ classлЎң мғқм„ұ нӣ„ м¶”нӣ„ мӢңлӮҳлҰ¬мҳӨ ліҖкІҪ лҳҗлҠ” лЎңм§ҒмғҒ нҷ•мһҘм„ұмқҙ н•„мҡ”н•ң кІҪмҡ° interfaceлЎң ліҖкІҪн•ҳлҠ” кІғмқҙ мўӢлӢӨкі  мғқк°Ғн•©лӢҲлӢӨ.

к·ёл ҮлӢӨл©ҙ interfaceлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°лҠ” м–ҙл–Ө кІҪмҡ°м—җ мӮ¬мҡ©н•ҙм•јн• к№Ңмҡ”??

ліҙнҶө н•ҳлӮҳмқҳ кё°лҠҘм—җм„ң м—¬лҹ¬ кіімңјлЎң нҢҢмғқлҗҳлҠ” кІғмқ„ interfaceлЎң лӮҳлҲ„лҠ”кІҢ мўӢмқ„ кІғ к°ҷмҠөлӢҲлӢӨ.
мҳҲлҘј л“Өм–ҙ мҶҢм…ңлЎңк·ёмқё, нҢЁмҠӨмӣҢл“ң ліҖкІҪ(к°ңмқём •ліҙмҲҳм •, нҢЁмҠӨмӣҢл“ңм°ҫкё°), м•„мқҙл”” м°ҫкё°(нңҙлҢҖнҸ° мқёмҰқ, мқҙл©”мқј мқёмҰқ, кё°нғҖ л“ұл“ұ), м№ҙл“ң кІ°м ң(м№ҙл“ңмӮ¬ лі„ кІ°м ң м·ЁмҶҢ), кІҢмһ„(кІҢмһ„лі„ н”Ңл Ҳмқҙ, мў…лЈҢ)л“ұмқҙ мһҲмҠөлӢҲлӢӨ.
кіөнҶөм ҒмңјлЎң м“°мқҙлҠ” кё°лҠҘмқ„н•ҳлӮҳмқҳ кё°лҠҘм—җм„ң м¶©л¶„нһҲ нҷ•мһҘлҗ  мҲҳ мһҲлҠ” кІҪмҡ° interfaceлҘј мӮ¬мҡ©н•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

лЎңк·ёмқёмқҳ кё°лҠҘмқ„ л§Ңл“Ө л•Ң Spring Securityмқҳ OAuth2лҘј мқҙмҡ©н•ҳм—¬ ліҙнҶө кё°лҠҘ кө¬нҳ„мқ„ н•©лӢҲлӢӨ.

к·ёл Үм§Җл§Ң oauth2лҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  кё°лҠҘмқ„ кө¬нҳ„н•ҳлҠ” кІҪмҡ°лҘј мғҳн”Ң мҪ”л“ңлҘј мқҙмҡ©н•ҳм—¬ м„ӨлӘ…н•ҙліҙкІ мҠөлӢҲлӢӨ.

лЁјм Җ лЎңк·ёмқё мң нҳ•м—җ лҢҖн•ң м •мқҳлҘј н•©лӢҲлӢӨ.

public LoginService getLoginType(LoginDto.Request dto) {
  if (dto == null) {
    return null;
  }

  if (dto.getAuthProvider().equals(AuthProvider.github)) {
    return new LoginGithubService();

  } else if (dto.getAuthProvider().equals(AuthProvider.google)) {
    return new LoginGoogleService();

  } else if (dto.getAuthProvider().equals(AuthProvider.kakao)) {
    return new LoginKakaoService();

  } else if (dto.getAuthProvider().equals(AuthProvider.naver)) {
    return new LoginGoogleService();

  } else if (dto.getAuthProvider().equals(AuthProvider.local)) {
    return new LoginLocalService();
  }

  return null;
}
