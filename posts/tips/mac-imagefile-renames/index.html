Mac에서 이미지 파일 이름 일괄 변경하는 방법

6월 18, 2025 · 1 분
목차
  • 1. Finder를 이용한 기본적인 파일 이름 변경
  • 2. Automator를 이용한 고급 파일 이름 변경
    • Automator 실행 방법
    • 이름 변경 설정

윈도우에서는 PowerToys 프로그램을 이용하면 파일들의 이름을 쉽게 일괄 변경할 수 있습니다.

Mac에서도 비슷한 기능을 찾아보니, 기본 Finder 기능과 Automator를 이용한 두 가지 방법으로 파일 이름을 일괄 변경할 수 있더군요.

1. Finder를 이용한 기본적인 파일 이름 변경#

가장 간단한 방법은 Finder를 이용하는 것입니다.

위와 같이 이미지를 선택한 후 우클릭하여 ‘Rename…‘을 선택하면 아래와 같은 이름 변경 창이 나타납니다.

이 방법을 사용하면 선택한 파일들이 다음과 같이 자동으로 이름이 변경됩니다.

  • image1.png
  • image2.png
  • image3.png
  • …

2. Automator를 이용한 고급 파일 이름 변경#

만약 파일 이름을 ‘image01.png’, ‘image02.png’와 같이 두 자리 숫자로 변경하고 싶다면, Automator를 사용해야 합니다.

Automator 실행 방법#

  1. Spotlight 검색(Command + Space)에서 ‘Automator’를 검색하여 실행합니다.

  1. ‘Rename Finder Items’를 추가하면 두 개의 항목이 나타납니다.
    • 위 항목: 변경할 파일 목록
    • 아래 항목: 이름 변경 방식 선택

이름 변경 설정#

‘Make Sequential’ 설정을 사용하여 이름을 커스터마이즈할 수 있습니다.

  • 이름 형식: ’new-image 01.xxx’
  • 숫자 위치: 이름 끝에 추가
  • 결과: 파일들이 순서대로 변경

위의 설정 후 오른쪽 위의 재생 버튼을 누르면 자동으로 파일 이름이 변경됩니다.

이 방법을 사용하면 파일 이름을 더 손쉽게 관리할 수 있습니다. 특히 많은 수의 이미지 파일을 정리할 때 유용합니다.
